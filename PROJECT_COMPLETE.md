## 📞 SUPPORT & HELP

### Common Issues

**Q: 3D sphere not showing?**
A: Check WebGL support (https://get.webgl.org/) and update graphics drivers

**Q: Can't connect to backend?**
A: Make sure SETUP_3D is running or manually start backend with `python backend/app.py`

**Q: Form not submitting?**
A: Check browser console (F12) and ensure all fields are filled

**Q: Mobile not working?**
A: Try landscape mode or update your browser
