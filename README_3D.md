## 🔧 TROUBLESHOOTING

### 3D Not Showing?
- Update your graphics drivers
- Try Chrome browser
- Check WebGL support (https://get.webgl.org/)

### Backend Connection Error?
- Make sure SETUP_3D script is running
- Check port 5000 is available
- Look for "Backend server starting" message

### Form Not Submitting?
- Check browser console (F12)
- Make sure all fields are filled
- Check backend is running

### Mobile Not Working?
- Rotate to landscape mode
- Try landscape orientation
- Check device has WebGL support

## 💡 TIPS & TRICKS

### For Best Experience
- Use Chrome or Firefox
- Allow JavaScript
- Enable WebGL
- Full screen for 3D
- Portrait mode on mobile

### To Customize Colors
Edit `styles.css` and change:
```css
:root {
    --primary: #e74c3c;     /* Change this */
    --secondary: #3498db;   /* or this */
}
```

### To Change 3D Colors
Edit `3d-scene.js` and look for `updateRiskMeterColor()`
